# APA Coach v1.5 Upgrade Notes

## Summary

v1.5 keeps the Vercel-based Document Check workflow from v1.4 and focuses on trust, clarity, and professor-review readiness. The site is positioned as an APA learning prototype, not a final APA formatting checker.

## Main Changes

- Unified current version language as APA Coach v1.5 Review Build.
- Changed complete Build output status to "Draft citation - verify with APA guide."
- Blocked copy behavior for incomplete citations and replaced the output with a clear incomplete message.
- Added source-type field rationale so students know why required fields matter.
- Added clipboard error feedback when browser permissions block copying.
- Added a Document Check review-boundary notice explaining that spacing, margins, page numbers, title page layout, and hanging indentation still require manual review.
- Grouped Document Check feedback into Detected issue, Possible issue, and Manual review required.
- Added upload summary details: file name, type, size, parse status, and layout reliability.
- Treated PDF review as text extraction review by default.
- Collapsed optional BYOK LLM settings and strengthened API key warnings.
- Expanded Learn practice feedback with why, APA principle, and next-step guidance.
- Added Learn search and first-year assignment mistake guidance.
- Added a 3-step Faculty Review flow and a feedback template.

## Boundary Statement

This version can identify useful APA signals from extracted text and visible formatting hints. It cannot fully verify final APA layout requirements such as spacing, margins, page numbers, title page layout, hanging indents, or every source-type edge case.

## Recommended Faculty Review Question

Does v1.5 communicate its learning-tool boundaries clearly enough while still giving students useful citation, reference, and document-level feedback?

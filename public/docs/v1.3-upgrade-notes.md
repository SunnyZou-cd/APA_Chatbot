# APA Coach v1.3 Upgrade Notes

## Purpose

v1.3 responds to faculty feedback about two priorities:

- Improve APA reference formatting feedback, especially author initials and italics.
- Move toward full-document APA review while keeping the current static-site privacy boundary.

## Reference Formatting Improvements

v1.3 adds rule-based feedback for compressed author initials. For example, `Lacy, J.T.` is flagged and explained as needing a space between initials: `Lacy, J. T.`

Build now separates plain text citation output from visual formatting. Students can still copy a plain text reference, while the page visually marks source-type-specific italics:

- Journal article: journal title and volume.
- Book/report/webpage/video/software-like standalone work: work title.
- Edited book chapter: edited book title.

Check also gives source-type-specific italics reminders so students learn why the italicized element changes by source type.

The Check input now accepts rich-text paste from tools such as Word or Google Docs. Italics, bold text, and basic font information remain visible in the input box, while rule-based diagnosis still uses the extracted text content.

## Document Check Prototype

Document Check supports:

- `.docx` files exported from Google Docs or Word.
- `.pdf` files exported from Google Docs or Word.
- `.txt` files.
- Pasted text.

The check runs in the browser session. The app does not add Google login, does not connect to Google Docs, does not add a backend, and does not store uploaded files.

## What Document Check Can Flag

The prototype can identify likely issues in extracted text:

- Missing References section.
- No clear APA-style in-text citations.
- Possible in-text citation and reference-list mismatch.
- Compressed author initials.
- DOI or URL format issues.
- Possible title-case issues in reference titles.
- Need to review italics based on source type.

## What Still Needs Manual Review

Some APA formatting details are layout-sensitive and cannot be guaranteed from extracted text:

- Title page placement and required title-page elements.
- Font, margins, and double-spacing.
- Hanging indents in the reference list.
- APA heading levels, centering, bolding, and capitalization.

The app marks these as manual review items instead of pretending they are final automatic validation.

## Recommended Faculty Review Question

Does this v1.3 prototype provide a useful bridge toward full-document APA review while clearly communicating that layout-sensitive formatting still requires human review in Google Docs or Word?

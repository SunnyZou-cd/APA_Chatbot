# APA Coach v1.6 Upgrade Notes

## Summary

v1.6 modernizes the complete APA Coach interface while preserving the learning-first behavior and Vercel Document Check API from v1.5. The release uses a calm forest palette and a web-based Liquid Glass treatment designed for readability across desktop, tablet, and mobile screens.

## Main Changes

- Applied the exact `#12372A`, `#436850`, `#ADBC9F`, and `#FBFADA` palette across the application.
- Added layered translucent surfaces, inner highlights, tinted shadows, and solid-color fallbacks.
- Added a desktop sidebar that can collapse to an icon rail and remembers the preference locally.
- Added an automatic compact rail for medium screens and a fixed bottom navigation bar for phones.
- Restyled all four tools: Build, Document Check, Learn, and Faculty Review.
- Added a skip link, stronger keyboard focus states, active-page semantics, reduced-motion support, and reduced-transparency support.
- Added updated page metadata, a favicon, and a valid `robots.txt` file.

## Behavior Preserved

The release does not change citation generation, document parsing, API request formats, upload handling, BYOK settings, or the four main navigation labels. Uploaded documents are still parsed for the current request only and are not stored by the app.

## Quality Results

- 117 automated tests passed.
- Client and API TypeScript builds passed.
- Dependency audit found zero vulnerabilities.
- Lighthouse scored 100 for Performance, Accessibility, Best Practices, and SEO.
- Largest Contentful Paint measured 1.4 seconds and Cumulative Layout Shift measured 0 in the local production audit.

## Boundary Statement

APA Coach remains a learning tool, not a final APA validator. Spacing, margins, page numbers, title-page layout, hanging indents, source truth, and unusual source-type edge cases still require manual review.

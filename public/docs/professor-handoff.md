# APA Coach v1.0 Faculty Review Notes

## Short Description

APA Coach v1.0 is a learning-first APA support prototype. It is not a paper generator. It helps students understand citations, references, APA setup expectations, and academic integrity boundaries.

Live review URL:

```text
https://apa-chatbot.vercel.app
```

Students can use it to:

- Build a reference entry with matching in-text citation forms.
- Diagnose a citation or reference attempt.
- Receive hint-first feedback before revealing a suggested correction.
- Review APA rule cards, paper setup checks, AI citation guidance, and short practice prompts.

## Current Boundaries

Version 1.0 does not include:

- Student accounts.
- A database.
- Live AI API calls.
- Student text storage.
- Full-paper uploads.
- Paper drafting or argument rewriting.

## Suggested Review Focus

Please review:

- Whether the tool feels like a tutor instead of an answer generator.
- Whether the academic integrity boundary is visible and clear.
- Whether the APA examples and warnings fit psychology coursework.
- Whether AI citation and disclosure language should be adjusted for course policy.
- Whether the Build, Check, Learn, and Faculty Review sections are appropriate for student use.

## Recommended Walkthrough

1. Open the site.
2. Review the journal article example in Build.
3. Delete the year or DOI and confirm the warning is clear.
4. Open Check, use the sample citation, and select Check citation.
5. Read the hint-first feedback, then reveal one suggested correction.
6. Open Learn and review the rule cards and practice prompts.
7. Open Faculty Review and evaluate the delivery and privacy language.

## Possible Next Decisions
